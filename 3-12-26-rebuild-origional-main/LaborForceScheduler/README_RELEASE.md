# LaborForceScheduler (Release Run Notes)

## Quick start

### Windows
- Double-click `run_scheduler.bat`, or run:
  - `py scheduler_app_v3_final.py`

### macOS / Linux
- Run:
  - `python3 scheduler_app_v3_final.py`
  - or `./run_scheduler.sh`

## Runtime folders
The app auto-creates required runtime folders if missing:
- `data/`
- `data/final_schedules/`
- `data/backups/`
- `exports/`
- `history/`
- `state_laws/`
